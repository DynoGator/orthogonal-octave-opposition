# Orthogonal Octave Opposition (OOO)
**Phase-Asymmetric Field-Induced Mass Transport**

This repository contains the validation materials for **Orthogonal Octave Opposition (OOO)** — an impedance-based method for non-contact mass transport and selective separation in symmetric fluid systems.

## Key Findings
* **Net Mass Transport:** Observed solely under phase asymmetry ($91.2^\circ$).
* **Selectivity:** Separation efficiency correlates with dielectric relaxation time ($\tau$), allowing for inertial filtering of $H_2O$ from hydrated ions.
* **Thermodynamics:** Operates as a non-equilibrium steady state system (NESS).
